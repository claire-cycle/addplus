https://github.com/plwanfeng/addplus

## 功能特点

- 命令行界面，无需GUI
- 多账户支持，可配置多个Cookie轮流使用
- 多线程处理，大幅提高涨分效率
- 自动获取最新用户名数据
- 详细的日志输出

## 使用方法

### 1. 配置账户

在`accounts.json`文件中配置您的账户信息，格式如下：

```json
[
  {
    "name": "账户1",
    "cookie": "您的完整cookie字符串"
  },
  {
    "name": "账户2",
    "cookie": "另一个cookie字符串"
  }
]
```

### 2. 运行程序

基本用法：

```
python addplus_cli.py
```

高级用法：

```
python addplus_cli.py --accounts 自定义账户文件.json --threads 10
```

参数说明：
- `-a, --accounts`: 指定账户配置文件路径，默认为`accounts.json`
- `-t, --threads`: 指定线程数量，默认为5

### 3. 中断程序

在运行过程中，按`Ctrl+C`可以安全地中断程序执行。

## 注意事项

- 线程数不宜设置过高，建议根据您的网络情况和账户数量调整
- 确保账户配置文件中的Cookie格式正确且有效
- 程序会自动保存和更新用户名数据到`client_username.json`文件
